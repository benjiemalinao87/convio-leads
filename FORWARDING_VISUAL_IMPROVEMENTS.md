# Forwarding System - Visual Improvements Proposal

## Current Problems

### 1. **Too Complex to Understand**
- Two-level control (master + rule toggles)
- Three matching criteria (Product AND Zip AND State)
- Priority system (lower = higher)
- Forward modes (first-match vs all-matches)
- Wildcard support

### 2. **Not Visual**
- Only tables and forms
- No flow diagrams
- No visual rule builder
- No visual representation of how rules work
- No visual testing/simulation
